Parser for [YouTrack grammar](https://www.jetbrains.com/help/youtrack/standalone/Search-Query-Grammar.html)

Hot Keys
-
* ⌃+z - cancel
* ⌃+⇧+z - cancel canceling
* ⌥+↑ - select node
